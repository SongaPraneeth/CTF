A musician left us a message. What's it mean?

Steps to solve:

step1: Use command `cat message.txt` to inside the file

step2: Now we will find some numbers in brackets, if we search this we will know that these are co-ordinates

step3: We put these coordinates in Google Maps in the form of (longitude, latitude). We observe that we get a city for each coordinate pair.
